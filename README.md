# 🌟 About

**VyaparAI** is a modern, AI-powered **Business Management Platform** designed for MSMEs, retailers, wholesalers, distributors, manufacturers, and service businesses.

It combines billing, GST invoicing, inventory, accounting, CRM, sales analytics, expense tracking, reporting, and AI-powered business insights into a single enterprise dashboard inspired by modern business software. The platform is designed as a frontend-first architecture that is ready for integration with any backend services.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/SurajBais8/VyaparAI.git
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Production Build

```bash
npm run build
```

Preview

```bash
npm run preview
```
